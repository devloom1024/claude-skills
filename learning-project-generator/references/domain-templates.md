# 学习领域模板

本文件定义不同学习领域的定制化模式。分析内容时，识别领域并应用相应的定制。

## 领域检测模式

**投资/理财：**
- 关键词：投资组合、股票、债券、理财规划、财富、退休、资产配置
- 书籍示例：《永久投资组合》《聪明的投资者》《漫步华尔街》
- 定制重点：投资组合追踪、市场分析、风险评估

**编程/算法：**
- 关键词：算法、编程、数据结构、刷题、代码、软件开发
- 网址特征：leetcode.com、labuladong.online、codeforces.com
- 定制重点：代码题解、刷题追踪、模式识别

**职业认证：**
- 关键词：考试、认证、CFP、CPA、CFA、备考、考点
- 特征：章节列表、练习题、知识点权重
- 定制重点：知识点掌握度、考试倒计时、模拟测试

**语言学习：**
- 关键词：语法、词汇、口语、听力、流利度、语言课程
- 特征：课程、对话、练习
- 定制重点：词汇表、发音笔记、对话练习

**学术科目：**
- 关键词：教材、课程、讲座、作业、大纲
- 特征：章节结构、习题集
- 定制重点：作业追踪、概念图、公式表

**技能/手艺：**
- 关键词：教程、指南、技巧、实践、手工
- 示例：烹饪、摄影、音乐、木工
- 定制重点：项目作品集、技法笔记、进度照片

## 混合/自定义领域

当内容不适合单一领域或高度专业化时：

1. **识别主要领域**（主要学习目标是什么？）
2. **识别次要方面**（涉及哪些辅助技能？）
3. **组合相关定制**（从多个领域中提取）
4. **添加自定义目录/章节**（针对独特内容）

示例："Python 量化金融"
- 主要：金融
- 次要：编程
- 自定义补充：/backtests/、/models/、/data-analysis/
- 追踪维度：投资概念 + 编程技能

## 内容特定适配

除领域模式外，还需考虑内容结构：

**书籍结构：**
- 按章节：按章节完成度追踪
- 按主题：按概念掌握度追踪
- 按项目：按项目完成度追踪

**课程结构：**
- 按周进度：按周/模块追踪
- 自定进度：按主题完成度追踪
- 集训营式：包含截止日期和里程碑

**教程系列：**
- 顺序式：追踪线性进度
- 选择路径式：追踪多条路径
- 参考式：追踪常用模式

## 实施要点

生成学习项目时：

1. **深入分析内容** - 不仅匹配关键词，要理解学习目标
2. **选择主要领域** - 选择最适合的领域模板
3. **智能定制** - 根据具体内容特征调整模板
4. **添加独特元素** - 包含针对此特定内容的目录/追踪
5. **解释选择** - 在生成的 README.md 中解释为何选择此结构
